# WEB-Interia

MVP Fáza 1: Laravel 12 prezentačný web s Bootstrap 5, GDPR/Cookies consent bannerom a Docker prostredím.

## Lokálne spustenie (Docker)

```bash
docker compose up --build
```

Aplikácia: http://localhost:8000

## Hlavné stránky

- `/` domov
- `/o-nas`
- `/produkty-sluzby`
- `/kontakt`

## SEO a verejné súbory

- `public/robots.txt`
- `public/sitemap.xml`

## Testy

```bash
php artisan test
```
